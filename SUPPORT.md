# Support

## Where to Ask Questions
- Use GitHub Issues for bugs and feature requests
- Use Discussions (if enabled) for general questions and ideas

## What to Include
Please include:
- Hardware: Arduino model, ESP32 model
- Sensor: MAX30102/MAX30105
- Connection method: UART baudrate, BLE UUIDs
- iOS device model + iOS version
- Logs and screenshots if possible

## Response Expectations
This is an open-source project. Responses are best-effort and may take a few days.
